# GeekInk Workspace Specification

## Product Name
GeekInk Workspace

## Product Type
Community-first learning workspace for GeekInk students, mentors, and admins.

## Core Idea
GeekInk Workspace is not a traditional LMS. It is a lightweight digital workspace for announcements, materials, assignments, submissions, mentorship, notifications, and community engagement.

## Main Users

### Student
- View announcements
- Access materials
- Submit assignments
- Receive mentor feedback
- Participate in community posts
- Track progress

### Mentor
- Create announcements
- Review submissions
- Give feedback
- Manage cohort communication
- Highlight student progress

### Admin
- Manage users
- Manage cohorts
- Manage mentors
- Manage materials
- Monitor platform activity

## MVP Features

### Authentication
- Register
- Login
- Logout
- JWT-based sessions
- Role-based access

### Cohorts
- Create cohort
- Assign mentor
- Enroll students
- View cohort dashboard

### Announcements
- Global announcements
- Cohort announcements
- Read/unread state
- Push notification support

### Materials
- Upload files
- Share links
- Organize by cohort
- Organize by module/topic

### Assignments
- Create assignment
- Set deadline
- Attach resources
- Publish to cohort

### Submissions
- Submit GitHub link
- Submit file link
- Track status
- Receive mentor feedback

### Community Feed
- Student posts
- Comments
- Reactions
- Mentor highlights

### Notifications
- In-app notifications
- Mobile push notifications
- Assignment reminders
- Announcement alerts

## Non-Goals
- Full LMS complexity
- Quiz engine
- SCORM support
- Video conferencing
- Complex grading system
- Microservices
- AI grading in MVP

## Product Principle
Keep the system simple, human, mobile-first, and focused on communication, mentorship, and accountability.
